My Portfolio

Creative Meme Generator App

Users signup and post memes users to rate. With this app you can generate pictures from the imgflip API, add text to the pictures easily and save your work. It's built with a Ruby on Rails backend and a React.js frontend. Project website

Rate Your Professor App

This app enables users to signup, add professors to a database and rate them. It's built with a Postgres database, on a Node.js-Express backend with a Vue.js frontend and a bootstrap framework. Project website

Capstone Project

Inspired by the book ​How To Talk So Kids Will Listen & Listen So Kids Will Talk​, this app provides a system for logging SMART goals, tracking progress and a motivational bar graph. It's built on a Ruby on Rails backend and Vue.js frontend and incorporates a Bootstrap framework and the Chart.js and Moment.js libraries.
